# GPS Location Matcher

## Files in this folder

- `matcher.py` — Python script that matches GPS points based on distance
- `points_a.csv` — First list of GPS coordinates (latitude, longitude)
- `points_b.csv` — Second list of GPS coordinates (latitude, longitude)
- `matches.csv` — Output file generated by the script (created after running)
- `test_matcher.py` — Unit tests
- `.github/workflows/ci.yml` — Runs tests automatically on GitHub

## What this program does

This program takes two lists of GPS locations and for each point in `points_a.csv`,
finds the closest point in `points_b.csv` using the Haversine distance formula.

## How to run


Run the script from the folder:

```bash
python3 matcher.py
```

## Output

- The closest matches are printed to the terminal
- A file called `matches.csv` is created with the results

## CSV format

Both CSV files must have the following headers:

```
lat,lon
```

Each row represents one GPS location.
